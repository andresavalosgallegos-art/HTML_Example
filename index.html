Cómo convertir un archivo .py a .exe (y crear un instalador)

Paso 1: Código de ejemplo (calculadora simple)

Usaremos el siguiente código en calculadora.py:

def suma(a, b):
    return a + b

def resta(a, b):
    return a - b

def multiplicacion(a, b):
    return a * b

def division(a, b):
    if b == 0:
        return "No se puede dividir por cero"
    return a / b

print("Calculadora simple")
print("1. Suma\n2. Resta\n3. Multiplicación\n4. División")
op = input("Elige una opción: ")

a = float(input("Primer número: "))
b = float(input("Segundo número: "))

if op == "1":
    print("Resultado:", suma(a, b))
elif op == "2":
    print("Resultado:", resta(a, b))
elif op == "3":
    print("Resultado:", multiplicacion(a, b))
elif op == "4":
    print("Resultado:", division(a, b))
else:
    print("Opción no válida")
Descargar calculadora.py

Paso 2: Instalar PyInstaller

Abre tu terminal o CMD y ejecuta:

pip install pyinstaller

Paso 3: Convertir el .py en .exe

En la terminal, ubícate en la carpeta donde está calculadora.py y ejecuta:

pyinstaller --onefile calculadora.py
  • Esto creará una carpeta dist con el ejecutable calculadora.exe.

Paso 4: (Opcional) Crear un instalador con Inno Setup

  1. Descarga e instala Inno Setup desde aquí.
  2. Crea un script básico como este (setup.iss):
; Script de ejemplo para Inno Setup
[Setup]
AppName=CalculadoraSimple
AppVersion=1.0
DefaultDirName={pf}\CalculadoraSimple

[Files]
Source: "dist\calculadora.exe"; DestDir: "{app}"; Flags: ignoreversion

[Icons]
Name: "{desktop}\Calculadora Simple"; Filename: "{app}\calculadora.exe"
  1. Abre el script en Inno Setup y haz clic en “Compile”.
  2. ¡Listo! Obtendrás un instalador que copia tu EXE a la PC del usuario.

Resumen

  1. Escribe el script Python (calculadora.py).
  2. Compílalo con PyInstaller (pyinstaller --onefile calculadora.py).
  3. Usa Inno Setup para crear el instalador.
