Cómo convertir un archivo .py a .exe y crear un instalador

Guía paso a paso para compilar tu aplicación Python (con GUI) en un ejecutable de Windows usando **PyInstaller** y generar un instalador con **Inno Setup**.

Paso 1: Código de ejemplo (Calculadora simple)

Usaremos el siguiente código Python, guárdalo como calculadora.py:

import tkinter as tk
from tkinter import messagebox

def click_boton(valor):
    actual = str(entrada.get())
    entrada.delete(0, tk.END)
    entrada.insert(0, actual + valor)

def limpiar():
    entrada.delete(0, tk.END)

def calcular():
    try:
        resultado = eval(entrada.get()) 
        entrada.delete(0, tk.END)
        entrada.insert(0, str(resultado))
    except:
        messagebox.showerror("Error", "Operación inválida")

ventana = tk.Tk()
ventana.title("Calculadora GUI")
ventana.geometry("300x400")

entrada = tk.Entry(ventana, width=16, font=("Arial", 24), borderwidth=2, relief="solid", justify="right")
entrada.grid(row=0, column=0, columnspan=4, padx=10, pady=10)

botones = [
    "7", "8", "9", "/",
    "4", "5", "6", "*",
    "1", "2", "3", "-",
    "0", ".", "=", "+"
]

fila = 1
col = 0

for boton in botones:
    if boton == "=":
        tk.Button(ventana, text=boton, width=5, height=2, font=("Arial", 18),
                  command=calcular).grid(row=fila, column=col, padx=5, pady=5)
    else:
        tk.Button(ventana, text=boton, width=5, height=2, font=("Arial", 18),
                  command=lambda b=boton: click_boton(b)).grid(row=fila, column=col, padx=5, pady=5)
    col += 1
    if col > 3:
        col = 0
        fila += 1

tk.Button(ventana, text="C", width=22, height=2, font=("Arial", 18), command=limpiar).grid(row=5, column=0, columnspan=4, padx=5, pady=5)

ventana.mainloop()
Descargar calculadora.py

Paso 2: Instalar PyInstaller

Abre tu terminal o CMD y ejecuta:

pip install pyinstaller

Paso 3: Convertir el .py en .exe

En la terminal, ubícate en la carpeta donde está calculadora.py y ejecuta el comando de compilación (usamos --noconsole para evitar la ventana de comandos):

pyinstaller --onefile --noconsole calculadora.py
  • Esto creará una carpeta dist con el ejecutable calculadora.exe.

Paso 4: (Opcional) Crear un instalador con Inno Setup

  1. Descarga e instala Inno Setup desde aquí.
  2. Crea un script básico como este (setup.iss) en la raíz del proyecto:
; Script de ejemplo para Inno Setup
[Setup]
AppName=CalculadoraSimple
AppVersion=1.0.0
DefaultDirName={autopf}\CalculadoraSimple
OutputDir=.\InstallerOutput

[Files]
Source: "dist\calculadora.exe"; DestDir: "{app}"; Flags: ignoreversion

[Icons]
Name: "{autodesktop}\Calculadora Simple"; Filename: "{app}\calculadora.exe"
  1. Abre el script en Inno Setup y haz clic en “Compile”.
  2. ¡Listo! Obtendrás el instalador en la carpeta `InstallerOutput`.

⚠️ Solución de Errores y Verificación de PyInstaller

Si la instalación en el **Paso 2** falló, revisa los siguientes problemas comunes.

1. Error: pip no se reconoce (Problema de PATH)

Si recibes 'pip' is not recognized... o pip: command not found, significa que el sistema no sabe dónde está la herramienta pip.

Solución:

  • Windows: Probablemente olvidaste marcar la casilla **"Add Python to PATH"** durante la instalación de Python. La solución ideal es reinstalar Python marcando esa opción.
  • Linux/macOS: A veces se necesita usar el gestor de paquetes de la versión específica de Python. Prueba: 
    pip3 install pyinstaller

2. Error: Permisos Denegados

Si recibes un error de **Permission denied**, `pip` está intentando escribir en una carpeta protegida por el sistema operativo.

Solución:

  • Recomendado (Usuarios): Instala PyInstaller solo para tu usuario (no requiere permisos de administrador): 
    pip install --user pyinstaller
  • Alternativa (Admin): Ejecuta la terminal **como administrador** (Windows) o usa `sudo` (Linux/macOS): 
    sudo pip install pyinstaller

✅ Comprobar que PyInstaller está instalado

Después de cualquier instalación exitosa, verifica que el comando esté disponible ejecutando:

pyinstaller --version

Si la instalación fue correcta, la terminal mostrará el número de versión (ej: 5.13.0). Si falla, debes solucionar los errores de PATH (punto 1) primero.

Resumen

  1. Escribe el script Python (calculadora.py).
  2. Compílalo con PyInstaller (pyinstaller --onefile --noconsole calculadora.py).
  3. Usa Inno Setup para crear el instalador.
