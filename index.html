Cómo convertir un archivo .py a .exe y crear un instalador

Guía paso a paso para compilar tu aplicación Python (con GUI) en un ejecutable de Windows usando **PyInstaller** y generar un instalador con **Inno Setup**.

Paso 1: Código de ejemplo (Calculadora simple)

Usaremos el siguiente código Python, guárdalo como calculadora.py:

import tkinter as tk
from tkinter import messagebox

def click_boton(valor):
    actual = str(entrada.get())
    entrada.delete(0, tk.END)
    entrada.insert(0, actual + valor)

def limpiar():
    entrada.delete(0, tk.END)

def calcular():
    try:
        # Nota: Usar eval() puede ser peligroso con entradas no controladas. 
        # Para un ejemplo simple está bien.
        resultado = eval(entrada.get()) 
        entrada.delete(0, tk.END)
        entrada.insert(0, str(resultado))
    except:
        messagebox.showerror("Error", "Operación inválida")

ventana = tk.Tk()
ventana.title("Calculadora GUI")
ventana.geometry("300x400")

entrada = tk.Entry(ventana, width=16, font=("Arial", 24), borderwidth=2, relief="solid", justify="right")
entrada.grid(row=0, column=0, columnspan=4, padx=10, pady=10)

botones = [
    "7", "8", "9", "/",
    "4", "5", "6", "*",
    "1", "2", "3", "-",
    "0", ".", "=", "+"
]

fila = 1
col = 0

for boton in botones:
    if boton == "=":
        tk.Button(ventana, text=boton, width=5, height=2, font=("Arial", 18),
                  command=calcular).grid(row=fila, column=col, padx=5, pady=5)
    else:
        tk.Button(ventana, text=boton, width=5, height=2, font=("Arial", 18),
                  command=lambda b=boton: click_boton(b)).grid(row=fila, column=col, padx=5, pady=5)
    col += 1
    if col > 3:
        col = 0
        fila += 1

tk.Button(ventana, text="C", width=22, height=2, font=("Arial", 18), command=limpiar).grid(row=5, column=0, columnspan=4, padx=5, pady=5)

ventana.mainloop()
Descargar calculadora.py

Paso 2: Instalar PyInstaller

Abre tu terminal (CMD, PowerShell o Bash) y ejecuta el siguiente comando para instalar PyInstaller:

pip install pyinstaller

Paso 3: Convertir el .py en .exe

En la terminal, navega hasta la carpeta donde guardaste calculadora.py y ejecuta el comando de compilación:

pyinstaller --onefile --noconsole calculadora.py
  • La bandera **`--onefile`** empaqueta todo en un solo ejecutable.
  • La bandera **`--noconsole`** evita que se abra la ventana de la terminal detrás de la GUI.
  • El ejecutable **`calculadora.exe`** se encontrará en la nueva carpeta **`dist`** que se creará.

Paso 4: (Opcional) Crear un instalador con Inno Setup

  1. Descarga e instala **Inno Setup** desde el sitio oficial: jrsoftware.org/isinfo.php.
  2. Crea un script de instalación (guárdalo como **setup.iss**) en la raíz de tu proyecto. Asegúrate de que apunte correctamente al archivo dentro de la carpeta `dist`:
; Script de ejemplo para Inno Setup
[Setup]
AppName=Calculadora Simple
AppVersion=1.0.0
Publisher=Tu Nombre o Empresa
DefaultDirName={autopf}\Calculadora Simple
Compression=lzma
SolidCompression=yes
OutputDir=.\InstallerOutput ; Directorio donde se guarda el instalador

[Files]
; Asegúrate de que el Source apunta al .exe creado por PyInstaller
Source: "dist\calculadora.exe"; DestDir: "{app}"; Flags: ignoreversion

[Icons]
; Crea un acceso directo en el escritorio y en el menú de inicio
Name: "{group}\Calculadora Simple"; Filename: "{app}\calculadora.exe"
Name: "{autodesktop}\Calculadora Simple"; Filename: "{app}\calculadora.exe"

[Run]
Filename: "{app}\calculadora.exe"; Description: "Lanzar Calculadora Simple"; Flags: postinstall skipifsilent
  1. Abre este archivo **`setup.iss`** en Inno Setup y haz clic en el botón **"Compile"**.
  2. El instalador final (`setup.exe`) se generará en la carpeta `InstallerOutput` (o el directorio que hayas especificado).

Resumen del Proceso

  1. Escribe el script Python (calculadora.py).
  2. Compílalo con PyInstaller: **`pyinstaller --onefile --noconsole calculadora.py`**.
  3. Usa Inno Setup para crear el instalador con un script `.iss`.
